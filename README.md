# Titanic Dataset - Exploratory Data Analysis (EDA)

Overview-

This project is part of my Data Science Internship at SkillCraft Technology.

The goal of this task was to perform data cleaning and exploratory data analysis (EDA) on the Titanic dataset using Python.

Tools & Libraries Used-

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

Tasks Performed-

* Loaded and explored the dataset
* Checked missing values
* Filled missing data using median and mode
* Removed unnecessary columns
* Created visualizations to analyze:
  * Survival count
  * Survival by gender
  * Passenger class distribution
  * Survival by passenger class
  * Age distribution
  * Correlation heatmap

Key Insights-
* Female passengers had a higher survival rate
* First-class passengers survived more often
* Age distribution showed most passengers were young adults
